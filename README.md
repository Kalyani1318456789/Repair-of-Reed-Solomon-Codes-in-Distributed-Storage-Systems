REED-SOLOMON ERASURE CODE IMPLEMENTATION(C)

OVERVIEW:
This Project implements a Reed-Solomon(n,k) erasure code over a finite field GF(p).where:n=p+1

The Program :
1. Generate a Random message of length k.
2. Encodes it into a codeword of length n with a polynomial function.
3. Randomly introduces erasures.
4. Recovers the Original message using Lagrange Interpolation.
5. Recovers the full CodeWord.

This demonstrates the mathematical foundation of ERROR-CORRECTING CODES used in:
* Distributed Storage Systems
* QR codes
* RAID systems
* Cloud storage redundancy

MATHEMATICAL BACKGROUND:

A message is related as coefficients of a polynomial:
            f(x) = m0​+m1​x+m2​x^2+⋯+mk−1​x^k−1

The CodeWord is generated by evaluating the polynomial at x = 1,2,3,...n
            c[i] = f(i+1) mod p

A degree k-1 polynomial is uniquely determined by k distinct points
Therfore:
* A code can tolerate upto n-k erasures.
* As long as k valid points remain, full recovery is possible.

HOW IT WORKS:

1.Encoding:
* Generate random coefficients m0,...,mk-1.
* Evaluate polynomial at points 1 to n.
* store Results in c[].

2.Erasure Simulation:
* Randomly remove upto n-k symbols.
* Erased symbols are marked as -1. 

3.Decoding:
* Collect first k non-erased points.
* Apply Lagrange interpolation.
* Recover Polynomial coefficients.
* Re-evaluate polynomial at all points to rebuild full codeword.

FUNCTIONS EXPLAINED:

power(base, exp, p) -->

Efficient modular exponentiation using binary exponentiation.
Time Complexity : O(log exp)

modInverse(a, p) -->

Computes modular inverse using Fermat's Little
Theorem:

a^-1 = a^p-2 mod p
! Requires p to be prime

evaluate (m, k, x, p) -->

evaluates polynomial at point x.
O(k)

create_random_erasures(c, n, k) -->

Randomly removes upto n-k values.

lagrange_evaluate(c, n, k) -->

Evaluates polynomial at a given point using Lagrange formula
(Not used in final reccovery, but demonstrates interpolation logic)

decode_and_recover( c, n, k) -->

Core decoder:
* collects k valid points.
* Constructs full polynomial via Lagrange Expansion.
* Recovers message coefficients.
* Recomputes full codeword.

TimeComplexity: O(k^3) (naive polynomial multiplication)

IMPORTANT CONSTRAINTS:

1. p = n+1 must be prime
2. k <= n
3. Erasures <= n-k
